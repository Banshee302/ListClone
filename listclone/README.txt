List-Clone is a lightweight software library, being built for creating hashes and creating compressed lclone files.

what is a .lclone file?
Lclone files or Listclone files are compressed read only files built to be quickly decompressed and or quickly being copied to different directories.

lclone files can be big in size especially for storing system modules and system services but the files are fast when it comes to moving to directories or extracting / compressing.
